# Security Policy

## Supported Versions

Only the latest version of NammaArasu on the `master` branch is currently actively supported for security updates.

| Version | Supported |
| ------- | --------- |
| >=0.2.x | Yes       |
| <0.2.x  | No        |

## Reporting a Vulnerability

We take the security of NammaArasu (நம்ம அரசு) seriously. If you discover a security vulnerability, we appreciate your help in disclosing it to us responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please report security issues by emailing us at **security@nammaarasu.org**.

Please include the following details in your report:
- Type of issue (e.g., buffer overflow, SQL injection, RLS bypass)
- Step-by-step instructions to reproduce the issue
- Any potential impact or exploit scenarios
- A proof-of-concept (PoC) script or screenshots, if available

### Our Response Process

1. **Acknowledgment**: We will acknowledge receipt of your report within 48 hours.
2. **Investigation**: Our team will investigate the issue and verify the vulnerability.
3. **Resolution**: We will work on a fix and coordinate a release.
4. **Advisory**: A security advisory will be published once the fix is deployed.

Thank you for helping keep our civic community safe!
